# fastapi-crud-project
A simple FastAPI-based CRUD REST API demonstrating clean backend architecture, database integration, and API design using Python.'
**Features**
- Create, Read, Update, Delete (CRUD) operations
- FastAPI framework
- SQLAlchemy ORM
- SQLite database
- Pydantic schemas
- Interactive Swagger UI
